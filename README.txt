Упрощённые шахматы.

- Компьютер делает ходы за обоих игроков;
- Рандомный выбор фигур для хода;
- Любой из возможных ходов для конкретной фигуры;
- Если фигура не имеет допустимых ходов - переключение на другую фигуру;
- Фигура не должна выходить за пределы доски или съедать своих;
- Условия завершения программы: у одного из игроков закончились доступные ходы или количество бессмысленных ходов больше определённого значения (>= 1000);
- Каждая фигура является классом, реализующим общий интерфейс (Chessman);
- Для перемещения фигур по шахматной доске используется класс (Mover) оперирующий фигурами через интерфейс Chessman;
